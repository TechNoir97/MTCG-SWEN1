Unittesting descicions:
Meine Unittests behandelnd hauptsächlich String manipulation. Angefangen von Json handling mit jackson oder Tests ob Strings auch richtig eingelesen werden,
und in Listen gespeichert werden.


Unique Feature:
Mein Unique Feature besteht daraus das ungebrauchte oder ungewollte Karten gescrapt werden können um für den User einen Nutzen aus
schlechten Karten ziehen zu können. Wenn ein User eine Karte scrapped so wird diese von seinem Deck entfernt und quasi "zerstört".
Der User erhält dafür eine Coin. Hintergedanke ist der, das man einen ungewollte Karte loswerden kann und trotzdem auch wenn geringen Wert daraus
ziehen kann.
